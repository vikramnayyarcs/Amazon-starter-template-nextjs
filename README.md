# Next JS ECommerce CLONE

ECommerce Store using Next JS, Stripe and FireBase.

## Installation Steps

### Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


### Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```

## How to Run

FrontEnd:

```npm run dev```

WebHook Listener:

``` stripe listen --forward-to localhost:3000/api/webhook ```
